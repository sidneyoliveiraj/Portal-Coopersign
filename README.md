# Portal-Coopersign
Sistema de Orquestração de Produção para Comunicação Visual

Sistema de Orquestração de Produção para Comunicação Visual, voltado para a cooperativa parceira Coopersign. A proposta é uma plataforma web que digitaliza e organiza o fluxo de pedidos, realizando validação técnica dos arquivos, aprovação do cliente antes da impressão, separação por tipo de máquina e organização da fila de produção por prioridade e prazo. O sistema atualiza o status em tempo real, calcula automaticamente metragem e valores, registra histórico de retrabalho e permite ao cliente acompanhar seus pedidos e histórico de serviços. Para a cooperativa, oferece painel operacional com gargalos e indicadores por máquina, material e período, apoiando a gestão e a tomada de decisão.
